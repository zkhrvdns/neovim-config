## Installation

### Install

1. Download appimage [NVIM v0.7.0](https://github.com/neovim/neovim/releases/tag/v0.7.0)

2. Rename it and set execution permission

   ```
   mv nvim.appimage nvim
   chmod u+x nvim
   ```

3. Place it in your PATH

   ```
   echo $PATH
   sudo mv nvim /usr/bin
   ```
